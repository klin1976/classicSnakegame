## Gameplay

- Move on a 20x20 grid.
- Eat red food to grow and score.
- Avoid walls, yourself, and obstacles (when enabled).

## Controls

- Keyboard: Arrow keys or `W/A/S/D`
- Buttons: `Start` / `Pause` / `Restart` / `Demo`

## Run locally

```bash
cd F:\Codex\classicSnakegame
python -m http.server 8000
```

Open: `http://localhost:8000`
