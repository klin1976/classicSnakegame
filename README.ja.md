## 遊び方

- 20x20 のグリッドでヘビを操作。
- 赤いエサを食べると成長して得点。
- 壁・自分の体・障害物（有効時）を避ける。

## 操作

- キーボード: 矢印キー または `W/A/S/D`
- ボタン: `Start` / `Pause` / `Restart` / `Demo`

## ローカル実行

```bash
cd F:\Codex\classicSnakegame
python -m http.server 8000
```

開く: `http://localhost:8000`
