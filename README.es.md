## Cómo jugar

- Mueve la serpiente en una cuadrícula 20x20.
- Come comida roja para crecer y sumar puntos.
- Evita paredes, tu cuerpo y obstáculos (si está activado).

## Controles

- Teclado: flechas o `W/A/S/D`
- Botones: `Start` / `Pause` / `Restart` / `Demo`

## Ejecutar localmente

```bash
cd F:\Codex\classicSnakegame
python -m http.server 8000
```

Abrir: `http://localhost:8000`
