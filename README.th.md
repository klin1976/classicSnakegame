## วิธีเล่น

- ควบคุมงูบนตาราง 20x20
- กินอาหารสีแดงเพื่อเพิ่มความยาวและคะแนน
- หลีกเลี่ยงกำแพง ตัวเอง และสิ่งกีดขวาง (เมื่อเปิดโหมด)

## การควบคุม

- คีย์บอร์ด: ปุ่มลูกศร หรือ `W/A/S/D`
- ปุ่ม: `Start` / `Pause` / `Restart` / `Demo`

## รันในเครื่อง

```bash
cd F:\Codex\classicSnakegame
python -m http.server 8000
```

เปิด: `http://localhost:8000`
