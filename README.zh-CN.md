# 经典贪吃蛇（Classic Snake）

使用纯 HTML / CSS / JavaScript 实现的模块化贪吃蛇游戏。

## 玩法

- 在 20x20 网格中移动蛇。
- 吃红色食物可增长并加分。
- 避开墙壁、自身和障碍物（开启障碍模式时）。

## 控制

- 键盘：方向键或 `W/A/S/D`
- 按钮：`Start` / `Pause` / `Restart` / `Demo`

## 本地运行

```bash
cd F:\Codex\classicSnakegame
python -m http.server 8000
```

打开：`http://localhost:8000`
