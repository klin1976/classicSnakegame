# 클래식 스네이크 (Classic Snake)

순수 HTML / CSS / JavaScript로 만든 모듈형 스네이크 게임입니다.

## 게임 방법

- 20x20 그리드에서 뱀을 이동합니다.
- 빨간 먹이를 먹으면 길어지고 점수가 오릅니다.
- 벽, 자기 몸, 장애물(활성화 시)을 피하세요.

## 조작

- 키보드: 방향키 또는 `W/A/S/D`
- 버튼: `Start` / `Pause` / `Restart` / `Demo`

## 로컬 실행

```bash
cd F:\Codex\classicSnakegame
python -m http.server 8000
```

열기: `http://localhost:8000`
